# Política de Privacidad


## 1. Información que recopilamos
- La aplicación **no recopila datos personales**.
- Las  capturas o imagenes de los códigos QR que el usuario guarda **se almacenan localmente** en el dispositivo.
- No se recopila información del dispositivo ni de navegación.

## 2. Uso de la información
Los QR almacenados tienen como único propósito facilitar la organización y permitir recibir pagos de manera rápida.

## 3. Permisos utilizados
La aplicación puede solicitar permisos como:
- **Acceso al almacenamiento** (para guardar y mostrar imágenes QR).

No usamos permisos adicionales ni accedemos a datos fuera de los estrictamente necesarios.
